Install using bower
bower install zeldicons

After installing with bower, just link the ‘bower_components/zelda.css’ file to your index file and you’re ready for adventure.

or Download Files
Download Zeldicons.zip

Unzip this file into your main directory (next to your index.html) then link the ‘zelda.css’ file to your index file and you’re ready to save Hyrule.

Equip your Icons
class=”link equip-tunic”

After installing (using either method, just add the class ‘zelda’ along with the icon specific class.
